# Agence Varek

Site web pour Agence Varek - Stabilisez vos revenus à 200 000 FCFA/mois minimum.

## Description

Site web responsive optimisé pour mobile, permettant aux utilisateurs de contacter l'agence via WhatsApp pour découvrir une méthode de génération de revenus.

## Fonctionnalités

- Design responsive (mobile uniquement)
- Intégration WhatsApp
- Interface moderne avec Bootstrap
- Optimisé pour la production

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Build de production

```bash
npm run build
```

## Hébergement

Le dossier `dist/` contient les fichiers optimisés pour la production. Vous pouvez l'héberger sur :

- Netlify
- Vercel
- GitHub Pages
- Tout serveur web statique

## Configuration

- Numéro WhatsApp : Modifiez `WHATSAPP_NUMBER` dans `src/App.jsx`
- Message WhatsApp : Modifiez `WHATSAPP_MESSAGE` dans `src/App.jsx`

## Technologies

- React 19
- Bootstrap 5
- Vite
- CSS Variables pour les couleurs
